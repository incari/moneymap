# Calculadora de Préstamo vs Inversión

## Descripción

Esta aplicación web es una herramienta interactiva que permite a los usuarios comparar los costos y beneficios de obtener un préstamo hipotecario para comprar una propiedad frente a invertir en un fondo indexado. La calculadora ofrece una visión detallada de los gastos asociados con la compra de una propiedad y los posibles rendimientos de una inversión a largo plazo.

## Características

- Calculadora de hipoteca/préstamo
  - Cálculo del pago mensual de la hipoteca
  - Estimación de costos iniciales y anuales
  - Proyección del valor de la propiedad a lo largo del tiempo
- Calculadora de fondo indexado
  - Cálculo del valor total de la inversión a lo largo del tiempo
  - Estimación de los intereses generados
- Visualización de resultados con gráficos interactivos
- Soporte multilingüe (Español e Inglés)

## Tecnologías utilizadas

- React
- Next.js
- TypeScript
- Tailwind CSS
- Recharts (para la visualización de datos)
- shadcn/ui (para componentes de UI)
